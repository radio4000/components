• This component displays if there is curently authentication, a user signed in or signed out
  • It can be used with children slot elements slot="in" and slot="out", for displaying content only when "signed in" or "signed out".
  • The auth attribute and property can be used to set the authentication status.
  • The component automatically updates `auth` listening to Supabase auth changes.
